# Ponder Redesign

Local project scaffold for redesigning <https://ponder.ing>.

The redesign implementation has not started yet; this repo only contains a ready-to-run Vite + React + TypeScript skeleton.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Notes

- Source app entry: `src/main.tsx`
- Styles entry: `src/styles.css`
- Design notes can go in `docs/`
